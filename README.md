# MAAUN-24-SWE-0105-Muhammad-Tukur-Bello#                                                                                                Mini CBT Engine
## What this app does
This is a simple Computer-Based Test (CBT) system built with Flask.
Users can answer questions and get their score with a timestamp.

## How to run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Features
- OOP using Question class
- Stack used to track answered questions
- datetime used for timestamp
- Web interface with Flask
